# NotesApp

Приложение заметки.

Функционал:
• Создание нескольких заметок в приложении;
• Вывод списка существующих заметок;
• Возможность редактирования любой заметки из списка;
• Удаление заметок;
• Сохранение заметок между сеансами.

Технологический стек:

• UIKit
• Realm
• Вёрстка кодом
• Схема MVC
