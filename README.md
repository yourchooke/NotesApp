# NotesApp

Приложение заметки.

Функционал: <br> 
• Создание нескольких заметок в приложении; <br> 
• Вывод списка существующих заметок; <br> 
• Возможность редактирования любой заметки из списка; <br> 
• Удаление заметок; <br> 
• Сохранение заметок между сеансами. <br> 

Технологический стек:

• UIKit <br> 
• Realm <br> 
• Вёрстка кодом <br> 
• Схема MVC <br>
